# Specialized queries

| query            | desc                                               | demo |
| ---------------- | -------------------------------------------------- | ---- |
| distance_feature |                                                    |      |
| more_like_this   | 查找与指定文本、文档或集合的相似文档               |      |
| percolate        |                                                    |      |
| rank_feature     | 一个基于数值特征值计算分数的查询                   |      |
| script           | 参考function_score                                 |      |
| script_score     | 允许使用脚本修改子查询分数的查询。                 |      |
| wrapper          | 接受为json或yaml字符串的其他查询。                 |      |
| pinned           | 将选定的文档置于匹配给定查询的其他文档之上的查询。 |      |
